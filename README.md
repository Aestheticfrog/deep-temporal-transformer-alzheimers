# deep-temporal-transformer-alzheimers
This project proposes a Deep Temporal Transformer (DTT) architecture for automated Alzheimer’s disease detection using MRI scans.  The model combines: - ResNet34 for spatial feature extraction - Transformer Encoder for temporal MRI sequence learning - Brown Bear Optimization (BBO) for feature-dimension tuning  f1 score 0.82
dataset link :-
https://www.kaggle.com/datasets/marcopinamonti/alzheimer-mri-4-classes-dataset
